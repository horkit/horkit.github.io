МЕТЕО.ХОРК
| Специальные лабораторные установки

Специальные лабораторные установки МЕТЕО.ХОРК представлены линейкой оборудования на базе общепромышленных испытательных систем со специализированным назначением (исполнение для прикладной области применения).

Оборудование значительно различается в зависимости от функционального назначения и области применения.



  • | Установка лабораторная МЕТЕО.ХОРК ТЕСТ-БЕТОН

    Лабораторные установки МЕТЕО.ХОРК ТЕСТ-БЕТОН предназначены для моделирования климатических условий при определении прочности бетонов по контрольным образцам (ГОСТ 10180-2012 Бетоны. Методы определения прочности по контрольным образцам).



  • | Установка лабораторная МЕТЕО.ХОРК АУСТЕНИТ

    Лабораторные установки МЕТЕО.ХОРК АУСТЕНИТ предназначены для использования на этапах термобработки сталей для устранения остаточного аустенита (значительного превращения аустенита в мартенсит).


  • | Установка лабораторная МЕТЕО.ХОРК КОНВЕКЦИЯ

    Лабораторные установки МЕТЕО.ХОРК КОНВЕКЦИЯ предназначены для использования в технологических процессах термообработки деталей из металлов и сплавов.


  • | Установка лабораторная МЕТЕО.ХОРК МАГНИЙ

    Лабораторные установки МЕТЕО.ХОРК МАГНИЙ предназначены для проверки отливок из магниевых сплавов на воздействие повышенной относительной влажности.



  • | Установка лабораторная МЕТЕО.ХОРК РАДИАЛ-ТЕСТ

    Лабораторные установки МЕТЕО.ХОРК РАДИАЛ-ТЕСТ предназначены для моделирования климатических условий при автоматических функциональных тестах партии электронных компонентов.



  • | Установка лабораторная МЕТЕО.ХОРК РОСТОК

    Лабораторные установки МЕТЕО.ХОРК РОСТОК предназначены для моделирования климатических условий при выращивании растений.


  • | Установка лабораторная МЕТЕО.ХОРК ЗЕРНО

    Лабораторные установки МЕТЕО.ХОРК ЗЕРНО предназначены для моделирования климатических условий при селекции семян.


  • | Установка лабораторная МЕТЕО.ХОРК МОРОЗ

    Лабораторные установки МЕТЕО.ХОРК МОРОЗ предназначены для моделирования климатических условий при определении морозо- и зимостойкости сельскохозяйственных культур.



  • | Установка лабораторная МЕТЕО.ХОРК МЕДИКАЛ

    Лабораторные установки МЕТЕО.ХОРК МЕДИКАЛ предназначены для моделирования климатических условий при хранении биоматериала.


  • | Установка лабораторная МЕТЕО.ХОРК МЕДИКАЛ

    Лабораторные установки МЕТЕО.ХОРК МЕДИКАЛ предназначены для моделирования климатических условий при хранении биоматериала.



  • | Установка лабораторная МЕТЕО.ХОРК СМОЛА

    Лабораторные установки МЕТЕО.ХОРК СМОЛА предназначены для испытаний древесно-волокнистых плит (и дргуих материалов) при определении эмиссии формальдегида и других вредных летучих веществ (ГОСТ 30255-2014 Мебель, древесные и полимерные материалы. Метод определения выделения формальдегида и других вредных химических веществ в климатических камерах).



  • | Установка лабораторная МЕТЕО.ХОРК СТЕКЛОПАКЕТ-АВТО

    Лабораторные установки МЕТЕО.ХОРК СТЕКЛОПАКЕТ-АВТО предназначены для моделирования климатических условий при определении точки росы и влагоустойчивости стеклопакетов (ГОСТ 52172-2003 Стеклопакеты для наземного транспорта. Технические условия).


Оборудование в зависимости от количества позиций в отправке, региона и способа доставки, может быть упаковано в различные типы упаковки. При этом под упаковкой понимается комплекс операций, связанных с подготовкой к герметизации, защите, перемещению, доставке, хранению, транспортированию и демонстрации продукции.

Эксплуатационная документация из состава комплекта поставки соответствует требования отечественной нормативной документации (ГОСТ Р 2.601-2019 Эксплуатационные документы, ГОСТ Р 2.610-2019 Правила выполнения эксплуатационных документов).

Комплекты ЗИП могут значительно отличаться при поставке. Комплект запасных частей, инструментов и принадлежностей формируется, исходя из требований достаточности комплекта для выполнения мероприятий технического облуживания и ремонта оборудования на протяжении полного срока эксплуатации. При этом, если эксплуатирующее предприятие не проводит самостоятельно техническое облуживание и ремонт оборудования – комплект ЗИП может быть сформирован, исходя из миниального необходимого набора запасных частей.

В состав поставки может входить набор пользовательски программ, позволяющих выполнять операторские и сервисные функции с автоматизированного рабочего места оператора, а также диспетчерское управление испытательным оборудованием с настольного ПК (локально или удаленно).
